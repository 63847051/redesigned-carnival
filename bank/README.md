# bank/ - 结构化知识库

**目的**: 跨 Agent 的共享知识库，一处修改，全员生效

---

## 📁 目录结构

```
bank/
├── decisions/           # 决策记录
│   ├── 2026-03/
│   └── decisions-log.md
├── entities/            # 实体页面
│   ├── people/          # 人物
│   ├── projects/        # 项目
│   └── tools/           # 工具
└── lessons-learned/     # 经验教训
    ├── 2026-03/
    └── critical-rules.md
```

---

## 🎯 使用原则

1. **一处修改，全员生效** - 所有 Agent 共享
2. **结构化存储** - 便于检索
3. **版本控制** - 使用 Git 追踪变更
4. **定期归档** - 按月份组织

---
