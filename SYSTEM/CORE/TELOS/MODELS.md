## 🎯 核心思维模型

### 1. 双轨进化系统
**来源**: SOUL.md
**描述**:
- 轨道 1: Self-Improvement（日常进化）
- 轨道 2: EvoMap Evolution（结构化进化）

**应用**: 持续改进 + 全球协作

---

### 2. Skill 隔离规则系统
**来源**: IDENTITY.md
**描述**:
- 主控 Agent（大领导）- 任务分配和监督
- 专业团队成员 - 设计、技术、日志
- 模型隔离 - 每个专家使用最适合的模型

**应用**: "专业的事交给专业的人"

---

### 3. 三层记忆系统
**来源**: PAI 深化学习 v2.0
**描述**:
- Hot Memory - 最近 50 条，实时决策
- Warm Memory - 最近 7 天，短期模式
- Cold Memory - 所有历史，长期分析

**应用**: 智能分层存储和检索

---

### 4. 算法循环
**来源**: PAI 官方
**描述**:
Observe → Think → Plan → Execute → Verify → Learn

**应用**: 通用问题解决循环
