# 室内设计师进阶学习计划（2026-2027）

**学员背景**：
- 年龄：41 岁
- 经验：10+ 年大型办公内装经验
- 技能：CAD 熟练，Revit 会一点
- 目标：提升竞争力，转型/突破

**学习周期**：12 个月（2026.3 - 2027.3）

**核心目标**：
1. 熟练掌握 Revit（BIM 协调能力）
2. 掌握灯光设计（差异化竞争）
3. 了解机电暖通（项目统筹能力）
4. 建立个人品牌（作品集 + 线上展示）

---

## 📋 总体时间表

| 月份 | 重点学习内容 | 时间投入 | 预期成果 |
|------|-------------|---------|---------|
| **M1** | Revit 基础 + 灯光基础 | 10h/周 | Revit 基础建模，灯光理论 |
| **M2** | Revit 进阶 + 灯光实践 | 10h/周 | Revit 独立建模，简单灯光方案 |
| **M3** | Revit 高级 + 作品集 | 10h/周 | Revit 项目实战，作品集初版 |
| **M4** | 机电暖通基础 + 作品集 | 8h/周 | 看懂专业图纸，作品集完善 |
| **M5-M6** | 综合应用 + 项目实践 | 6h/周 | 完整项目案例 |
| **M7-M9** | 个人品牌 + 职业转型 | 4h/周 | 线上展示，简历优化 |
| **M10-M12** | 持续学习 + 职业发展 | 2h/周 | 稳定输出，持续成长 |

---

## 🎯 Month 1-3: Revit 系统学习

### 🎯 学习目标

**M1**: 建立基础
- 熟悉 Revit 界面和基本操作
- 掌握建模基本命令
- 完成简单办公室建模

**M2**: 进阶提升
- 掌握族（Family）创建
- 掌握明细表和视图
- 完成中型办公室建模

**M3**: 实战应用
- 掌握协作和协同
- 掌握施工图出图
- 完成完整办公楼项目

### 📚 学习资源

**线上课程**：
- **推荐**：筑龙网 Revit 系统课程（办公项目专项）
- **备选**：B站免费教程（搜索"Revit 办公"）
- **书籍**：《Revit 202X 建筑设计实战》

**练习项目**：
- M1: 小型办公室（200㎡）
- M2: 中型办公室（1000㎡）
- M3: 大型办公室（3000㎡）

### ⏰ 时间安排

**每周 10 小时**：
- **工作日**：每天 1 小时（20:00-21:00）
- **周末**：每天 2.5 小时（上午或下午）

**学习节奏**：
- 40% 看视频/看书
- 60% 实操练习

### 📊 检验标准

**M1 结束**：
- [ ] 能独立建立简单办公室模型
- [ ] 理解 Revit 的基本概念（族、类型、实例）
- [ ] 能创建基本视图

**M2 结束**：
- [ ] 能创建常用的族（门、窗、家具）
- [ ] 能创建明细表（门窗表、家具表）
- [ ] 能独立完成中型办公室建模

**M3 结束**：
- [ ] 能创建完整的办公楼项目
- [ ] 能出具施工图（平面图、立面图、剖面图）
- [ ] 理解协作流程

## 💡 Month 1-3: 灯光设计学习

### 🎯 学习目标

**M1**: 理论基础
- 理解光照基本概念
- 掌握照度计算方法
- 了解常用灯具类型

**M2**: 实践应用
- 掌握办公空间灯光标准
- 掌握灯光布局方法
- 完成简单灯光方案

**M3**: 进阶提升
- 掌握智能控制
- 掌握节能设计
- 完成完整灯光方案

### 📚 学习资源

**书籍**：
- 《照明设计手册》（基础）
- 《办公空间照明设计》（专业）

**线上**：
- 云知光照明学院（免费课程）
- 照明论坛（案例学习）

**练习**：
- M1: 小型办公室照度计算
- M2: 中型办公室灯光方案
- M3: 大型办公室智能照明

### ⏰ 时间安排

**每周 3 小时**（和 Revit 同步）：
- 周末：1.5 小时学习 + 1.5 小时练习

### 📊 检验标准

**M1 结束**：
- [ ] 理解照度、色温、显色指数
- [ ] 会计算办公室照度
- [ ] 了解常用灯具

**M2 结束**：
- [ ] 能做简单的灯光方案
- [ ] 会选型灯具
- [ ] 会做灯光布局

**M3 结束**：
- [ ] 能做完整的灯光方案
- [ ] 理解智能控制
- [ ] 能提供节能建议

## 🔧 Month 4: 机电暖通基础

### 🎯 学习目标

**M4**: 了解基础
- 看懂机电图纸
- 看懂暖通图纸
- 理解管线综合

### 📚 学习内容

**机电**：
- 强电系统（照明、插座）
- 弱电系统（网络、电话）
- 消防系统（报警、喷淋）

**暖通**：
- 空调系统（风管、设备）
- 新风系统（送风、排风）
- 消防排烟

**管线综合**：
- 常见冲突点
- 解决方案
- 协调方法

### 📚 学习资源

**不需要买书**，看：
- 项目图纸（你手上的项目）
- 供应商资料（设备厂家会提供）
- 网上案例

### ⏰ 时间安排

**每周 5 小时**：
- 周末：2.5 小时看图纸 + 2.5 小时整理笔记

### 📊 检验标准

**M4 结束**：
- [ ] 能看懂机电图纸
- [ ] 能看懂暖通图纸
- [ ] 知道常见冲突点
- [ ] 知道怎么协调解决

## 🎨 Month 3-4: 作品集制作

### 🎯 目标

**整理 10 年经验，展示核心竞争力**

### 📋 内容结构

**第 1 部分：个人介绍（1 页）**
- 基本信息
- 核心优势
- 职业定位

**第 2 部分：代表作品（5-8 个）**

每个项目包含：
- 项目概况（面积、业主、周期）
- 设计理念（1-2 段）
- 平面图 + 效果图（2-4 张）
- 竣工实景（2-4 张）
- 设计亮点（2-3 点）

**第 3 部分：技能展示**
- Revit 项目（M3 的作业）
- 灯光方案（M3 的作业）
- 施工图（你手上的项目）

**第 4 部分：联系方式**

### 🎨 格式

**PPT**：用于面试、会议
**PDF**：用于邮件发送
**线上**：Behance/站酷/个人网站

### ⏰ 时间安排

**M3**: 收集资料
**M4**: 整理制作

- 2 个完整的 BIM 项目案例

### M7-M9: 个人品牌 + 职业转型

**7.1 线上展示**
- 小红书：每周 1 个案例
- 站酷/Behance：每月 1 个项目
- 个人网站：展示作品集

**7.2 简历优化**
- 突出 10 年大型办公经验
- 突出 Revit + BIM 能力
- 突出灯光设计能力
- 突出项目协调能力

**7.3 职业定位**

**选项 A：BIM 协调/深化设计师**
- 目标公司：设计公司、工程公司
- 职位：BIM 经理、深化设计总监
- 薪资：15-25K

**选项 B：设计管理**
- 目标公司：甲方、地产公司
- 职位：设计经理、项目总监
- 薪资：20-30K

**选项 C：独立接项目**
- 目标：小型办公项目（改造、扩建）
- 收入：项目制，不稳定但自由

## 💪 执行建议

### 1. 立即开始

**本周就做**：
- [ ] 购买/找到 Revit 学习资源
- [ ] 购买/找到灯光学习资源
- [ ] 收集你过去项目的资料
- [ ] 制定每周具体时间表

### 2. 保持节奏

**不要中断**：
- 每周至少 6 小时学习
- 即使忙，也要学 2-3 小时
- 养成习惯，像吃饭一样

### 3. 实践为主

**学完就练**：
- Revit 学一个命令，就用一个命令
- 灯光学一个概念，就做一个练习
- 不要等"学完"再"用"

### 4. 记录进度

**每周总结**：
- 这周学了什么？
- 这周做了什么？
- 下周要做什么？
