## 🚀 启动命令

### 启动主面板
```bash
cd /root/.openclaw/workspace/dashboard
python3 -m http.server 8000
```

### 启动所有服务
```bash
# 主面板
cd /root/.openclaw/workspace/dashboard && python3 -m http.server 8000 &

# Zero Token（已在运行）
cd /root/.openclaw/workspace/openclaw-zero-token && ./server.sh start

# 股票分析（已在运行）
cd /root/.openclaw/workspace/stock-analysis-system && bash start.sh &

# AI Dashboard（已在运行）
cd /root/.openclaw/workspace/ai-team-dashboard && node server.js &
```
