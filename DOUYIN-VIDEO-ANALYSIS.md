**步骤 1**: 检查可用的服务器
```bash
mcporter list
# 当前只有: exa, cn-a-stock, weibo
```

**步骤 2**: 添加抖音 MCP 服务器

需要配置抖音的 MCP 服务器。这可能需要：
1. 抖音开放平台 API
2. MCP 适配器
3. 配置文件设置

**步骤 3**: 测试抖音读取

