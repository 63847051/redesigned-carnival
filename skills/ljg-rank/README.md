# ljg-rank 降秩引擎

给一个领域，找出背后真正撑着它的几根独立的力。

## 功能

- **降秩**：从复杂现象中找到独立生成器
- **四个判据**：生成性、最小性独立性、预测力
- **散文输出**：像讲故事一样展现降秩过程

## 使用方式

```
/ljg-rank 互联网行业
降秩 物理学
找秩 商业模式
```

## 核心概念

**秩不是关键要素，而是生成器。**

- 用生成器能反向生成全部现象
- 关掉任何一个就有现象解释不了
- 每对生成器能找到真实案例
- 能推导出原始清单之外的新现象

## 触发词

- 降秩
- 找秩
- 这个领域靠什么
- /ljg-rank
