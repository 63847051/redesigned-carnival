# ljg-learn 概念解剖

将任何概念从八个维度深度解剖，压成一句顿悟。

## 功能

- **定锚**：定义核心概念和常见误解
- **八刀**：8 维度全方位分析（历史、辩证、现象、语言、形式、存在、美感、元反思）
- **内观**：第一人称视角，找出深层结构
- **压缩**：压成一句顿悟

## 使用方式

```
/ljg-learn 熵
/ljg-learn 自由
概念解剖 民主
```

## 输出示例

```
#+title: 概念解剖：熵
#+filetags: :concept:
#+date: [2026-03-24]

* 定锚
** 通行定义：系统无序程度的度量
** 常见误解：熵只等于"混乱"

* 八刀
** 历史：...
** 辩证：...
...

* 内观
* 压缩
```
