# ljg-plain 白话引擎

把任何内容改写到聪明的十二岁小孩也能懂。

## 功能

- **白话改写**：复杂内容 → 通俗易懂
- **红线约束**：9 条硬性规则确保可读性
- **零术语**：12 岁孩子能复述

## 使用方式

```
/ljg-plain 量子力学
白话说 爱因斯坦相对论
说人话 区块链
```

## 核心规则

1. **口语检验**：读出声像跟朋友说话吗？
2. **零术语**：专业词必须先落地大白话
3. **短词优先**：能用两个字不用四个字
4. **一句一事**：每句只推进一步
5. **具体**：名词看得见，动词有力气
6. **开头给理由**：第一句让人想读下一句
7. **不填充**：每句都在干活
8. **信任读者**：说一遍够了
9. **诚实**：想不清楚就说想不清楚
