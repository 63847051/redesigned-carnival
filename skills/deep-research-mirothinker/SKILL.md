## 技能描述
深度研究助手。当用户需要深度研究、多方对比、交叉验证、体系化结论时使用此技能。会自动调用 MiroThinker 进行长链推理和网页探索，生成扎实的研究报告。

整合了李继刚"八刀"解剖法：对任何概念进行八个维度的深度解剖，最后压缩成一句顿悟。

## 触发场景
- 深度研究
- 调研分析  
- 多方对比
- 交叉验证
- 综合分析
- 系统性研究
- 概念解剖
- 解释/理解一个概念

## 核心功能
1. **识别深度研究需求** - 自动识别用户需要深度研究的意图
2. **自动调用 MiroThinker** - 通过 web_search 和推理能力进行深度分析
3. **获取研究报告** - 收集和分析多源信息
4. **转换为用户友好格式** - 生成结构化、易读的研究报告
5. **八刀解剖法** - 对核心概念进行八个维度的深度解剖
6. **压缩顿悟** - 将复杂理解压成一句核心洞察

## 使用方法
当用户提到深度研究相关需求时，此技能会被自动激活。用户无需手动指定技能名称。

## 八刀解剖法（整合自 ljg-learn）

对研究中的核心概念，使用八刀解剖法：

### 第一刀：历史
最早从哪冒出来 → 怎么变的 → 哪一步拐成了今天的意思

### 第二刀：辩证
它的反面是什么 → 正反碰撞后，更高一层的理解是什么

### 第三刀：现象
扔掉所有预设，回到事情本身 → 用一个日常场景把它还原出来

### 第四刀：语言
拆字源（中/英/希腊/拉丁）→ 画出相邻概念的语义网 → 这个词暗含什么隐喻

### 第五刀：形式
写一个公式或形式化表达 → 公式在哪里失效

### 第六刀：存在
这个概念改变了人怎么活着

### 第七刀：美感
它美在哪？用一个具体意象呈现

### 第八刀：元反思
我们在用什么隐喻理解它？这个隐喻挡住了什么？换一个会怎样

## 压缩理念

用一本书的厚度讲一个概念，最后压成一句顿悟。

### 压缩步骤
1. **公式**：`概念 = ...`
2. **一句话**：用最简单的话说出最深的理解
3. **结构图**：纯 ASCII 画出概念的骨架

## 研究流程

### Phase 1：定锚
1. 核心概念的最通行定义是什么？常见误解在哪？
2. 概念里藏着哪几个核心词素？

### Phase 2：八刀
八个方向各切一刀。每刀 2-3 句，只留筋骨。

### Phase 3：内观
1. 变成这个概念本身，用第一人称看世界。3-5 句。
2. 八刀之中，哪几刀指向同一个深层结构？把它提出来。

### Phase 4：压缩
1. 写公式
2. 说一句顿悟
3. 画结构图

## 技术实现
- **网页搜索**: 使用 web_search 获取最新信息
- **长链推理**: 通过多步骤推理构建深度分析
- **信息整合**: 跨源信息对比和验证
- **八刀解剖**: 八个维度深度解剖核心概念
- **压缩提炼**: 从海量信息中提取核心顿悟

## 输出格式

研究报告将包含以下部分：

### 标准结构
1. 研究概述
2. 关键发现
3. 多方对比分析
4. 交叉验证结果
5. 结论和建议
6. 参考资料

### 深度研究结构（概念类）
当研究涉及核心概念时，增加：

```
## 概念解剖
### 定锚
### 八刀
- 历史
- 辩证
- 现象
- 语言
- 形式
- 存在
- 美感
- 元反思
### 内观
### 压缩
- 公式
- 一句顿悟
- 骨架图
```

## 注意事项
- 适用于需要系统性分析的问题
- 会调用网络搜索获取最新信息
- 生成的报告需要用户根据实际需求调整
- 复杂研究可能需要较长时间完成
- 八刀解剖只用于核心概念，不需要对每个概念都用八刀
