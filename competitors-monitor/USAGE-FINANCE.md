# 财务报表分析 - 使用指南

## 概述

财务报表分析系统自动解析 PDF 财务报表，计算关键财务指标，与基准对比，发现异常情况。

## 功能特性

- **PDF 解析**: 自动提取财务报表中的关键数据
- **指标计算**: 计算盈利能力、流动性、杠杆率等指标
- **基准对比**: 与设定基准对比，发现差距
- **异常检测**: 自动识别财务风险

## 配置说明

编辑 `config/finance.json`:

```json
{
  "companies": [
    {
      "name": "示例公司A",
      "stock_code": "600000",
      "fiscal_year": 2025,
      "benchmark": {
        "roe": 0.15,
        "gross_margin": 0.30,
        "debt_ratio": 0.60
      }
    }
  ],
  "thresholds": {
    "debt_ratio_high": 0.70,
    "current_ratio_low": 1.0
  }
}
```

## 运行方式

```bash
cd /root/.openclaw/workspace/competitors-monitor
python3 plugins/finance.py
```

## 财务指标说明

### 盈利能力

| 指标 | 说明 | 计算方式 |
|------|------|----------|
| 毛利率 | 销售收入减去成本后的利润率 | (收入-成本)/收入 |
| 净利率 | 净利润占收入的比例 | 净利润/收入 |
| ROE | 股东权益回报率 | 净利润/股东权益 |
| ROA | 资产回报率 | 净利润/总资产 |

### 流动性

| 指标 | 说明 | 理想值 |
|------|------|--------|
| 流动比率 | 流动资产/流动负债 | > 1.5 |
| 速动比率 | (流动资产-存货)/流动负债 | > 1.0 |

### 杠杆率

| 指标 | 说明 | 阈值 |
|------|------|------|
| 负债率 | 总负债/总资产 | < 70% |
| 产权比率 | 总负债/股东权益 | < 1.0 |

## PDF 解析

将 PDF 财务报表放在 `data/financial_reports` 目录下，系统会自动解析。

支持提取的数据:
- 营业收入
- 净利润
- 毛利率
- 总资产
- 总负债
- 股东权益

## 集成数据源

### Tushare API

1. 注册 tushare.pro
2. 获取 Token
3. 配置:

```json
"api": {
  "enabled": true,
  "provider": "tushare",
  "token": "YOUR_TOKEN"
}
```

## 告警类型

| 类型 | 说明 |
|------|------|
| margin_decline | 毛利率下降 |
| high_debt | 负债率过高 |
| low_liquidity | 流动性不足 |
| benchmark_breach | 低于基准 |
