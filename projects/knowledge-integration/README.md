## 🚀 使用方法

### 基本使用

```python
# 创建系统
system = KnowledgeSystem()

# 添加文档
result = await system.add_document("docs/api.md")

# 搜索知识
results = await system.search("如何使用 API？")

# 查看结果
for r in results:
    print(f"{r['id']}: {r['content']}")
```
