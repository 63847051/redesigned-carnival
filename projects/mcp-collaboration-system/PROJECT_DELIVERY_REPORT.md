### 创建工作流
```python
from mcp_workflow import create_simple_workflow, Task

tasks = [
    Task("t1", "任务1", "agent_a"),
    Task("t2", "任务2", "agent_b", dependencies=["t1"]),
]

workflow = create_simple_workflow("my_wf", "我的工作流", tasks)
result = await workflow.execute()
```
