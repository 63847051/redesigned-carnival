## 🏗️ 系统架构 v7.1

### 5层清晰架构

```
1. UI 层 (system/1-ui-layer/)
   - 终端界面
   - 消息显示
   - 用户交互

2. CLI 层 (system/2-cli-layer/)
   - 启动入口
   - 参数解析
   - 配置加载

3. 编排层 (system/3-orchestration-layer/)
   - QueryEngine
   - 任务分配
   - Agent 协调

4. 执行层 (system/4-execution-layer/)
   - Agent Loop
   - 工具执行
   - 结果处理

5. 能力层 (system/5-capability-layer/)
   - 工具系统
   - 权限管理
   - 记忆系统
```
