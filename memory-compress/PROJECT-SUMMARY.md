## 🎯 项目概述

一个从 Claude Code Auto Compaction 借鉴设计的生产级记忆压缩工具，能够自动检测、智能压缩、保留关键信息。

### 核心特性

- ✅ 自动检测是否需要压缩
- ✅ 智能提取待办事项、关键决策、重要链接
- ✅ 生成结构化摘要（Markdown 格式）
- ✅ 保留最近的重要记忆
- ✅ 可配置的压缩策略

## 🚀 使用示例

### 基本使用

```python
from memory_compress import compact_messages

# 准备消息
messages = [...]  # 你的消息列表

# 执行压缩
result = compact_messages(messages)

# 查看结果
print(f"压缩掉: {result['removed_count']} 条")
print(f"节省: {result['tokens_saved']} tokens")
```

### 压缩 MEMORY.md

```python
from memory_compress.integration import compress_memory_file

# 压缩 MEMORY.md
result = compress_memory_file("/root/.openclaw/workspace/MEMORY.md")

if result["compressed"]:
    print(f"✅ 压缩成功！")
    print(f"   节省: {result['size_saved']} 字符")
```
