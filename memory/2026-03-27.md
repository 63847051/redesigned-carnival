## 📊 量化分析系统

### 核心功能

**6 个量化因子**：
1. 连续涨跌因子（20%）
2. 周内日历因子（25%）
3. 月度日历因子（20%）
4. 极端行情因子（15%）
5. 技术面因子（10%）
6. 板块因子（10%）

### 预测流程

```
输入：股票代码（如 SH600000）
  ↓
获取实时数据（MCP）
  ↓
解析技术指标（KDJ、MACD、RSI、布林带）
  ↓
计算因子概率（加权平均）
  ↓
生成预测方向和概率
  ↓
生成操作建议（仓位、止损、信号）
  ↓
保存分析报告
```

### 使用方法

```bash
# 量化分析
/root/.openclaw/workspace/scripts/stock-integration/analyze.sh -q SH600000

# 基本分析
/root/.openclaw/workspace/scripts/stock-integration/analyze.sh -b SH600000
```
