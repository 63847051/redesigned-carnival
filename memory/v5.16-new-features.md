## 🚀 四大核心更新

### 1️⃣ DP-006: 子 Agent Token 优化 ⭐ 最重要

**成果**：
- **65% Token 节省** 🎯
- 子 Agent: 4300 tokens → 1500 tokens
- 响应速度加快 33-40%
- 免费模型支持（opencode/minimax-m2.5-free）

**技术实现**：
```json
// openclaw.json
{
  "subagents": {
    "context": {
      "excludeFiles": [
        "SOUL.md",      // 排除身份文件
        "IDENTITY.md",
        "USER.md"
      ]
    },
    "timeout": 300  // 5 分钟超时保护
  }
}
```

**设计原则**：
1. 身份 vs 技能分离
2. 主从上下文分层
3. 最小化 Token 传递

**文档**: `.learnings/design-patterns/DP-006-subagent-token-optimization.md`
