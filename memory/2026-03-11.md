## ⚠️ 遇到的问题

### 问题 1：配置被自动修复
- **现象**: 添加 sessions.spawn 配置后被 doctor 命令删除
- **原因**: OpenClaw 2026.3.2 不支持此配置
- **解决**: 升级到 2026.3.8

### 问题 2：双重包管理器
- **现象**: npm 和 pnpm 都安装了 OpenClaw
- **原因**: Gateway 使用 pnpm，CLI 使用 npm
- **解决**: 同时升级两个

### 问题 3：飞书渠道限制
- **现象**: Thread bindings are unavailable
- **原因**: 飞书插件不支持 thread 模式
- **解决**: 使用其他渠道或保持当前配置

### 问题 4：版本命名混乱
- **现象**: SOUL.md 显示 v3.1，实际是 v6.0
- **原因**: 多个版本分支，文档未更新
