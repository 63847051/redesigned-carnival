## OpenClaw 工作区
- **位置**: `/root/.openclaw/workspace`
- **用途**: OpenClaw AI 框架的主要工作区
- **重要性**: 所有项目文件、配置、文档都在这里

## 文档位置
- **文档**: `/root/.openclaw/workspace/docs/`
- **脚本**: `/root/.openclaw/workspace/scripts/`
- **记忆**: `/root/.openclaw/workspace/memory/`

## 访问方式
用户通过飞书与 OpenClaw 交互，工作区包含：
- SOUL.md - 系统身份和版本
- IDENTITY.md - 角色定位
- AGENTS.md - 永久规则
- MEMORY.md - 长期记忆
