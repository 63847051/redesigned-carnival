## 🚀 快速开始

### 方式 1：直接克隆（推荐）

```bash
# 1. 克隆仓库
git clone https://github.com/63847051/redesigned-carnival.git
cd redesigned-carnival

# 2. 阅读部署指南
cat docs/DEPLOYMENT_GUIDE.md

# 3. 配置 API Keys
cat docs/SECRETS_SETUP.md

# 4. 启动系统
openclaw gateway start
```

### 方式 2：从 Release 下载

```bash
# 下载 v5.12.0
wget https://github.com/63847051/redesigned-carnival/archive/refs/tags/v5.12.0.tar.gz

# 解压
tar -xzf v5.12.0.tar.gz
cd redesigned-carnival-5.12.0
```
