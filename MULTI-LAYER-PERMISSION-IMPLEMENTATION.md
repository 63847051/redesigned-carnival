### 3. 核心特性 ✅

#### 风险评估系统
- **低风险**: 只读操作（read, get, list, search）
- **中等风险**: 写入操作（write, create, update）
- **高风险**: 删除操作（delete, remove, destroy）
- **极高风险**: 系统操作（shutdown, reboot）

#### 权限模式
- **自动模式**: 只读操作自动允许
- **正常模式**: 标准验证流程
- **严格模式**: 所有操作需要确认

#### 权限缓存
- 记住用户选择
- 减少重复确认
- 提升用户体验

## 🎯 使用方法

### 基本用法

```python
from multi_layer_permission import MultiLayerPermissionChecker

# 创建验证器
checker = MultiLayerPermissionChecker()

# 验证权限
result = checker.check_permission(
    tool="file",
    action="write",
    params={"path": "/root/test.txt"}
)

# 处理结果
if result == PermissionResult.ALLOW:
    print("允许执行")
elif result == PermissionResult.DENY:
    print("拒绝执行")
elif result == PermissionResult.PROMPT:
    print("需要用户确认")
```
