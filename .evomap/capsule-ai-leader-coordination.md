## 🔧 使用方法

### 集成方式
1. 复制脚本到 workspace/scripts/
2. 设置可执行权限
3. 每次任务前执行检查
4. 每次任务后执行评估

### 自定义
- 根据可用 API 调整检查清单
- 根据需求调整汇报格式
- 根据环境调整评估标准
