## 💡 使用建议

### 深度研究时
1. 先定锚：核心概念是什么？常见误解在哪？
2. 用八刀切：八个维度各来一刀
3. 压缩顿悟：压成公式 + 一句话 + 骨架图

### 复盘迭代时
1. 找模式
2. 定根因
3. 制定措施（每措施压成一句话）
4. 写入规则库
5. 提炼核心顿悟

### 写 Prompt 时
1. 参考 prompt-writing-guide.md
2. 问自己：能删哪个字？
3. 问自己：能用更短的词吗？
4. 问自己：能压成一句话吗？
