## ✅ 优化 1.3: 飞书通知集成

### 功能
创建飞书通知系统，实时发送告警

### 实施内容

1. **创建通知脚本**
   - `/root/.openclaw/workspace/scripts/feishu-notify.sh`
   - 支持多种告警类型
   - 卡片式消息

2. **集成到现有脚本**
   - 内存监控脚本已集成
   - 自动发送告警

3. **配置指南**
   - `/root/.openclaw/workspace/docs/feishu-notification-guide.md`
   - 完整的配置步骤
   - 故障排查指南

### 支持的告警类型

1. **内存告警**
   - 内存 > 80% 时通知
   - 自动发送

2. **服务告警**
   - 服务停止时通知
   - 即时告警

3. **进化完成**
   - 进化学习完成时通知
   - 发送摘要

4. **健康检查**
   - 定期健康检查报告
   - 可选启用

### 使用方法

```bash
# 1. 配置 Webhook URL
export FEISHU_WEBHOOK_URL="https://open.feishu.cn/open-apis/bot/v2/hook/xxx"

# 2. 测试通知
bash /root/.openclaw/workspace/scripts/feishu-notify.sh "测试" "测试内容"

# 3. 自动告警
# 内存监控会自动发送告警
```

### 配置步骤

1. 在飞书创建群聊
2. 添加自定义机器人
3. 获取 Webhook URL
4. 配置环境变量
5. 测试通知
