### 1.2 清理大目录（5 分钟）

#### 目标
释放磁盘空间，优化存储

#### 执行步骤
```bash
# 1. 分析大目录
du -sh /root/.openclaw/workspace/* | sort -hr

# 2. 检查 node_modules
find /root/.openclaw/workspace -name "node_modules" -type d -exec du -sh {} \; | sort -hr

# 3. 清理 npm 缓存
npm cache clean --force

# 4. 清理临时文件
find /tmp -name "*.tmp" -mtime +7 -delete
```

#### 预期结果
- ✅ 释放 100MB+ 空间
- ✅ 清理无用缓存
- ✅ 优化目录结构
