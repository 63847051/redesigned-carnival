# 🚨 关键规则：每次升级后必须更新版本号

**规则ID**: RULE-003
**优先级**: 🔴 最高（CRITICAL）
**创建时间**: 2026-03-15 23:22
**触发原因**: 重复性错误 - 版本号忘记更新

---

## 🎯 规则内容

### 每次系统升级后必须更新版本号

**位置**: `/root/.openclaw/workspace/SOUL.md`

**必须更新的字段**：
```markdown
**版本**: 5.XX.0
**最后升级**: YYYY-MM-DD HH:MM
```

---

### 第 2 次（2026-03-15 23:22）

## ✅ 正确流程

### 每次系统升级时
1. ✅ 完成升级
2. ✅ **立即更新 SOUL.md 版本号**
3. ✅ 推送到 GitHub
4. ✅ 验证版本号已更新
