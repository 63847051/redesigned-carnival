### 2. 歧义检测脚本（50 tokens）

**文件**: `scripts/ambiguity-detector.sh`

**用途**: 自动检测用户指令是否有歧义

**工作流程**:
```
用户指令 → 运行脚本 → 有歧义？只回答 → 无歧义？继续
```

**示例**:
```bash

bash ambiguity-detector.sh "方案 C，可以并行的处理任务吗"
# 输出: ⚠️ 检测到疑问句 → 只回答，不执行
```

**Token 成本**: 50 tokens/次（只在需要时运行）

---

### 3. 操作前检查脚本（100 tokens）

**工作流程**:

**示例**:
```bash

```

**Token 成本**: 100 tokens/次（只在重要操作时运行）
