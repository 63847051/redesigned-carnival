# 版本更新承诺 - 必须遵守

**创建时间**: 2026-03-18 23:06
**创建原因**: 用户批评我每次更新版本时总是遗漏更新文档

---

## 🎯 承诺

**每次更新版本到仓库时，必须同时更新以下文件：**

### 1. SOUL.md ✅
- [ ] 版本号（如：v5.17.0 → v5.18.0）
- [ ] 最后升级时间
- [ ] 新增功能列表

### 2. README.md ✅
- [ ] 标题版本号
- [ ] 副标题版本号
- [ ] 详细更新说明（如果是大版本更新）
- [ ] 快速下载链接（wget URL）
- [ ] AI 框架版本（技术栈部分）
- [ ] 系统状态版本
- [ ] 最终版本号（文档末尾）

### 3. CHANGELOG.md ✅
- [ ] 添加新版本条目
- [ ] 详细更新内容
- [ ] 发布日期

### 4. Git Tag ✅
- [ ] 创建版本标签（如：v5.18.0）
- [ ] 推送标签到 GitHub
