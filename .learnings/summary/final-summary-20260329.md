## 🎯 **记忆系统最佳实践**

### ✅ **DO（应该做的）**

1. **运行时使用 `.context/`**
   - 快速检索
   - Token 优化
   - 按需加载

2. **每天自动同步**
   - 设置 cron 任务
   - 每天 23:59 执行
   - 避免数据丢失

3. **定期检查一致性**
   - 使用 `--check` 命令
   - 确保数据一致

4. **使用统一入口**
   - 所有操作通过 MemoryManager
   - 避免直接调用子系统

### ❌ **DON'T（不应该做的）**

1. **不要手动编辑两个系统**
   - 容易导致不一致
   - 使用协调器

2. **不要忽略同步**
   - 可能丢失数据
   - 设置自动同步

3. **不要混淆用途**
   - `.context/` 是运行时
   - `memory/` 是长期存储
