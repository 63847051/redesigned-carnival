### 🚀 部署和使用

#### 快速启动
```bash
cd /root/.openclaw/workspace/webui-monitoring
./start-monitoring.sh
```

#### 访问界面
- **主界面**: http://localhost:8080
- **健康检查**: http://localhost:8080/health
- **配置信息**: http://localhost:8080/config.js
- **WebSocket**: ws://localhost:8080

#### 配置选项
```bash
export PORT=8081                    # 端口
export UPDATE_INTERVAL=1000         # 更新间隔
export MAX_CLIENTS=50              # 最大客户端数
export HEARTBEAT_INTERVAL=60000     # 心跳间隔
```
