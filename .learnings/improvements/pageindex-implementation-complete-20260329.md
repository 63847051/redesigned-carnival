## 💡 核心洞察

### 洞察 1: 自底向上是关键
> **"先处理子节点，再处理父节点。"**

- L0（叶子）→ L1 → L2 → L3（根）
- 保证每个层级都有完整的摘要
- 父节点基于子节点生成自己的摘要

### 洞察 2: 树形结构更强大
> **"树形结构比扁平结构更易理解和维护。"**

- 清晰的层级关系
- 完整的路径追溯
- 更精准的搜索

### 洞察 3: 渐进式优化是安全的
> **"先用规则实现，再逐步优化为 LLM。"**

- 快速实现（规则）
- 逐步优化（LLM）
- 回退机制（容错）

## 📝 使用指南

### 自动摘要生成
```bash
# 处理昨天的数据
bash /root/.openclaw/workspace/scripts/auto-summary.sh

# 处理指定日期
bash /root/.openclaw/workspace/scripts/auto-summary.sh --date 2026-03-28

# 处理指定月份
bash /root/.openclaw/workspace/scripts/auto-summary.sh --month 2026-03
```

### 记忆树操作
```bash
# 构建树
python3 /root/.openclaw/workspace/scripts/memory-tree.py build

# 可视化树
python3 /root/.openclaw/workspace/scripts/memory-tree.py visualize

# 搜索树
python3 /root/.openclaw/workspace/scripts/memory-tree.py search --query "项目"

# 统计信息
python3 /root/.openclaw/workspace/scripts/memory-tree.py stats
```

### 任务调度
```bash
# 启动调度器（每天凌晨 1 点自动运行摘要生成）
```
