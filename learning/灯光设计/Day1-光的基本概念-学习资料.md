## 📚 Part 1：理论基础（1小时）

### 1.1 光通量（Luminous Flux）

#### 定义
光通量是指**光源发出的光能量总量**，是人眼能够感知的可见光功率。

#### 单位
- **流明**
- 符号：lm

#### 理解要点
1. **光源的总量**：不管光照到哪里，光通量只表示光源本身发出的光总量
2. **与人眼有关**：只计算人眼能看到的可见光（380-780nm）
3. **灯泡参数**：买灯泡时看到的"800流明"就是光通量

#### 实际案例
```
普通蜡烛：约 13 lm
40W 白炽灯：约 325 lm
60W 白炽灯：约 800 lm
100W 白炽灯：约 1600 lm
5W LED灯：约 400-500 lm
12W LED灯管：约 1200 lm
```

#### 记忆口诀
> 光通量 = 光源发出的光总量（不管照哪里）

---

### 1.2 照度（Illuminance）

#### 定义
照度是指**单位面积上接收的光通量**，表示光照射到表面的密度。

#### 单位
- **勒克斯**
- 符号：lx
- 定义：1 lm/m² = 1 lx

#### 计算公式
```
照度(E) = 光通量(Φ) / 面积(A)
```

**示例**：
- 1000 lm 的灯，照射在 10㎡ 的地面上
- 照度 = 1000 / 10 = 100 lx

#### 理解要点
1. **与距离有关**：同样的灯，离得越近，照度越大
2. **与面积有关**：光通量不变，面积越大，照度越小
3. **人眼感受**：照度越高，感觉越亮

#### 办公空间照度标准（GB 50034-2013）
| 区域 | 照度标准 |
|------|---------|
| 普通办公室 | 300 lx |
| 高档办公室 | 500 lx |
| 会议室 | 300 lx |
| 接待区 | 200 lx |
| 走廊 | 100 lx |
| 档案室 | 200 lx |

#### 实际感受
```
满月夜：0.1 lx
路灯照明：5-30 lx
客厅照明：100-200 lx
办公室照明：300-500 lx
晴天户外：10,000-100,000 lx
```

#### 记忆口诀
> 照度 = 单位面积上的光（照在地上的亮不亮）

---

### 1.3 亮度（Luminance）

#### 定义
亮度是指**发光面在某个方向的明亮程度**，是人眼直接看到的发光强度。

#### 单位
- **坎德拉/平方米**
- 符号：cd/m²

#### 理解要点
1. **人眼直接感受**：亮度是眼睛看到的"明暗程度"
2. **与反射有关**：同样的照度，白色墙面比黑色墙面亮
3. **有方向性**：不同角度看，亮度可能不同

#### 为什么同样的照度，看起来不一样？

**示例**：
```
照度 = 300 lx
- 白色墙面（反射率 80%）：亮度高，感觉很亮
- 灰色墙面（反射率 40%）：亮度中，感觉一般
- 黑色墙面（反射率 10%）：亮度低，感觉很暗
```

#### 办公空间亮度推荐
| 区域 | 亮度范围 |
|------|---------|
| 工作区（桌面） | 100-300 cd/m² |
| 电脑屏幕 | 80-200 cd/m² |
| 环境墙面 | 50-150 cd/m² |
| 天花板 | 30-100 cd/m² |

#### 记忆口诀
> 亮度 = 眼睛看到的明暗（发光面亮不亮）

---

## 📊 三者关系总结

### 核心区别
| 概念 | 含义 | 单位 | 关注点 |
|------|------|------|--------|
| **光通量** | 光源发出的光总量 | lm (流明) | 光源本身 |
| **照度** | 单位面积接收的光 | lx (勒克斯) | 被照表面 |
| **亮度** | 人眼看到的明暗 | cd/m² | 视觉感受 |

### 通俗理解
```
光通量 = 灯泡本身有多亮
照度   = 地面被照得有多亮
亮度   = 眼睛感觉有多亮
```

### 实际案例串联
```
场景：你买了一盏 1200 lm 的 LED 灯管

光通量：1200 lm（灯管本身的参数）
照度  ：安装在 15㎡ 办公室，照度约 200-300 lx
亮度  ：白色桌面看起来很亮，黑色文件看起来暗
```
