## 🎯 学习建议

### 每日学习流程

1. **学习前（5分钟）**
   - 回顾前一天的内容
   - 看今日学习目标
   - 准备好学习工具

2. **学习中（90分钟）**
   - 阅读学习资料
   - 观看推荐视频
   - 做笔记和标记

3. **练习后（20分钟）**
   - 完成自测题
   - 检查答案
   - 记录错题

4. **总结时（5分钟）**
   - 填写进度跟踪表
   - 记录疑问点
   - 计划明天学习

### 学习时间建议

**固定时间**：每天晚上 8:00-10:00

**时间分配**：
- 理论学习：60分钟
- 练习题：30分钟
- 总结和计划：30分钟

### 学习方法

1. **理解重于记忆**
   - 先理解概念
   - 再记忆数据
   - 最后应用实践

2. **理论结合实践**
   - 学完理论马上做练习
   - 去实际空间观察照明
   - 测量照度和色温

3. **及时复习**
   - 每天学习前复习
   - 每周末总结本周
   - 用思维导图整理
