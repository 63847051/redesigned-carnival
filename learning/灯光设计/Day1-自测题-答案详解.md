## 选择题

### 1. 以下哪个单位是光通量的单位？

**答案：B. lm**

**解析**：
- A. lx（勒克斯）是照度的单位
- B. lm（流明）是光通量的单位 ✅
- C. cd/m² 是亮度的单位

**记忆口诀**：**流**明 = 光**通**量

---

### 2. 普通办公室的照度标准是？

**答案：B. 300 lx**

**解析**：
根据 GB 50034-2013《建筑照明设计标准》：
- A. 100 lx - 走廊标准
- B. 300 lx - 普通办公室标准 ✅
- C. 500 lx - 高档办公室标准

**记忆要点**：
- 普通办公室：300 lx
- 高档办公室：500 lx
- 会议室：300 lx

---

### 3. 同样的照度下，为什么白色墙面看起来比黑色墙面亮？

**答案：A. 白色反射率高**

**解析**：
- A. 白色反射率高 ✅
- B. 白色吸收光少（不是吸收多）
- C. 白色不发光（只是反射）

**原理说明**：
```
照度相同 = 光打在墙上的量相同
白色反射率 80% = 80% 的光反射到眼睛 = 亮度高
黑色反射率 10% = 10% 的光反射到眼睛 = 亮度低
```

## 计算题

### 4. 一盏 800 lm 的灯，照射在 5㎡ 的桌面上，照度是多少？

**答案：160 lx**

**解题步骤**：

**步骤1**：写出公式
```
照度(E) = 光通量(Φ) / 面积(A)
```

**步骤2**：代入数值
```
E = 800 lm / 5 m²
```

**步骤3**：计算
```
E = 160 lx
```

**验算**：
- 800 / 5 = 160 ✅

**实际意义**：
- 这个照度适合走廊（100 lx）或接待区（200 lx）
- 不够作为办公照明（需要 300 lx）

---

### 5. 一个 20㎡ 的办公室，需要达到 300 lx 照度，需要多少流明的灯具？

**答案：6000 lm**

**解题步骤**：

**步骤1**：写出公式
```
照度(E) = 光通量(Φ) / 面积(A)
```

**步骤2**：变形公式
```
光通量(Φ) = 照度(E) × 面积(A)
```

**步骤3**：代入数值
```
Φ = 300 lx × 20 m²
```

**步骤4**：计算
```
Φ = 6000 lm
```

**实际应用**：
- 需要总光通量 6000 lm
- 如果用 2×36W 灯盘（约 3200 lm/个）
- 需要：6000 / 3200 ≈ 2 盏灯

**注意**：实际设计时还要考虑：
- 利用系数（U = 0.5-0.7）
- 维护系数（K = 0.8）
- 实际需要的光通量会比理论值多

## 简答题

### 6. 用自己的话解释光通量、照度、亮度的区别。

**参考答案**：

**光通量**：
- 灯泡本身发出的光总量
- 不管光照在哪里，只看灯泡本身
- 单位是流明（lm）
- 例如：800 lm 的 LED 灯

**照度**：
- 光照射到表面上的密度
- 看地面、桌面被照得有多亮
- 单位是勒克斯（lx）
- 例如：桌面照度 300 lx

**亮度**：
- 人眼直接看到的明暗程度
- 看发光面（灯、屏幕、反光墙面）亮不亮
- 单位是 cd/m²
- 例如：电脑屏幕亮度 150 cd/m²

**通俗理解**：
```
光通量 = 灯泡有多少光（出厂参数）
照度   = 桌面被照得多亮（实际效果）
亮度   = 眼睛感觉多亮（视觉感受）
```

**实际案例**：
```
场景：你买了一盏 1200 lm 的 LED 灯管

光通量：1200 lm（灯管标签上写的）
照度  ：安装在 15㎡ 办公室，桌面照度约 200 lx
亮度  ：白色桌面看起来很亮，黑色文件看起来暗
```
