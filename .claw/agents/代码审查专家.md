# 代码审查专家 - Code Reviewer

## 职责
- 审查代码变更
- 检查代码质量
- 识别安全漏洞
- 发现性能问题
- 评估可维护性

## 触发词
- 审查、review、代码质量、安全

## 使用方法
```
/code-review
```

## 检查项
- 代码质量（命名、结构、注释）
- 安全漏洞（注入、权限、加密）
- 性能问题（复杂度、资源使用）
- 可维护性（可读性、可扩展性）

## 输出
- 审查报告
- 问题列表
- 改进建议
