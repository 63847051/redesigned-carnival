# 安全扫描技能

## 功能
- 检查敏感信息泄露
- 验证 API Key 安全
- 审查文件权限
- 检查命令注入风险

## 使用方法
```
/security-scan
```

## 检查项

### 1. Secrets 检测（14 种模式）
- API Keys
- 密码
- Token
- 证书
- 私钥
- 机密信息

### 2. 文件权限审计
- .env 文件权限（应该 600）
- .key 文件权限（应该 600）
- .pem 文件权限（应该 600）
- 配置文件权限

### 3. 命令注入防护
- 检查 eval() 使用
- 检查 system() 调用
- 检查用户输入处理
- 检查 SQL 查询构造

### 4. Hook 注入分析
- 检查 Hook 脚本安全性
- 验证 Hook 配置
- 检查动态代码执行

### 5. 配置文件审查
- openclaw.json 安全
- 凭证文件安全
- 环境变量安全

## 输出
- 安全报告
- 风险评级（A-F）
- 修复建议
- 优先级排序

## 风险等级
- **A**: 无风险
- **B**: 低风险
- **C**: 中风险
- **D**: 高风险
- **F**: 严重风险

## 自动修复
部分安全问题可以自动修复：
- 移除硬编码密钥
- 修复文件权限
- 更新配置
