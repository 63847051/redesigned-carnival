## 💡 量化分析原理

### 因子 1：连续涨跌因子（20%）

**原理**: 基于历史连续涨跌后的次日表现概率

**逻辑**:
- 大涨后（>3%）→ 可能回调（下跌概率 60%）
- 小涨后（1-3%）→ 可能盘整（下跌概率 45%）
- 平盘后（-1% ~ 1%）→ 可能选择方向（上涨概率 55%）
- 小跌后（-3% ~ -1%）→ 可能反弹（上涨概率 60%）
- 大跌后（<-3%）→ 可能超跌反弹（上涨概率 65%）

### 因子 2：周内日历因子（25%）

**原理**: 基于历史相同星期几的上涨概率

**逻辑**:
- 周一：上涨概率略高（52%）- 周一效应
- 周五：下跌概率略高（52%）- 获利了结
- 其他：无明显偏向（50%）

### 因子 3：月度日历因子（20%）

**原理**: 基于历史相同日历日的上涨概率

**逻辑**:
- 月初（1-5号）：上涨概率略高（53%）- 资金流入
- 月末（25-31号）：下跌概率略高（53%）- 资金紧张
- 其他：无明显偏向（50%）

### 因子 4：极端行情因子（15%）

**原理**: 基于历史极端行情后的次日表现概率

**逻辑**:
- 极端上涨（>5%）→ 大概率回调（下跌概率 60%）
- 极端下跌（<-5%）→ 大概率反弹（上涨概率 60%）
- 其他：无明显偏向（50%）

### 因子 5：技术面因子（10%）

**原理**: 基于均线、MACD、KDJ、RSI、布林带等技术指标

**逻辑**:
- KDJ > 80：超买，可能回调（下跌概率 60%）
- KDJ < 20：超卖，可能反弹（上涨概率 60%）
- MACD 金叉：看涨（上涨概率 55%）
- MACD 死叉：看跌（下跌概率 55%）
- RSI > 70：超买（下跌概率 60%）
- RSI < 30：超卖（上涨概率 60%）
- 突破上轨：可能回调（下跌概率 60%）
- 跌破下轨：可能反弹（上涨概率 60%）

### 因子 6：板块因子（10%）

**原理**: 基于板块新闻情绪分析

**逻辑**:
- 利好：上涨概率略高
- 利空：下跌概率略高
- 中性：无明显偏向

## 🧮 概率计算方法

### 加权平均

```python
总概率 = Σ(因子概率 × 权重)

最终方向 = 
  if 总概率 > 0.5: up
  elif 总概率 < 0.5: down
  else: neutral

置信度 = |概率 - 0.5| × 2
```

### 示例

```
连续涨跌: up × 55% × 0.20 = 0.110
周内效应: down × 52% × 0.25 = 0.130
月度效应: down × 53% × 0.20 = 0.106
极端行情: neutral × 50% × 0.15 = 0.075
技术面: neutral × 50% × 0.10 = 0.050
板块: neutral × 50% × 0.10 = 0.050

总概率 = 0.521
方向 = down (因为 0.521 > 0.5)
上涨概率 = 1 - 0.521 = 0.479 = 47.9%
置信度 = |0.521 - 0.5| × 2 = 0.042 = 4.2%
```

## ⚠️ 重要提示

### 1. 量化预测不是保证

- **60% 概率** 意味着仍有 **40% 失败可能**
- 历史表现不代表未来结果
- 市场充满不确定性

- **不要全仓**：建议仓位 20-50%
- **设置止损**：跌破止损位坚决离场
- **分散投资**：不要把所有资金投入一只股票
