### Day 1: 理论学习（周一）

#### 🌙 晚上（20:00-21:00）- 灯光设计

**任务 1.1: 学习光的基本概念**

**学习内容**：
- 光通量（Φ）- 光源发出的光总量（单位：流明 lm）
- 照度（E）- 单位面积的光通量（单位：勒克斯 lx）
- 亮度（L）- 人眼感受到的明亮程度（单位：坎德拉/㎡ cd/㎡²）

**关系公式**：
```
照度 (lx) = 光通量 (lm) / 面积 (m²)
```

**记忆方法**：
- 光通量 = 灯泡发出的光总量
- 照度 = 地面接收到的光
- 亮度 = 人眼看到的亮

**任务 1.2: 学习色彩学基础**

**学习内容**：
- **色温（CCT）** - 光的颜色（单位：开尔文 K）
  - 2700K：暖白光（温馨、舒适）
  - 3000K：暖白光（接待、前台）
  - 4000K：自然白（办公、会议）
  - 5000K：冷白光（专注、高效）

- **显色指数（CRI）** - 还原物体真实颜色的能力（0-100）
  - 80-90：一般显色（办公空间最低要求）
  - 90+：高显色（展示空间）
  - 95+：超高显色（博物馆）

**练习**：
- 观察你家里的灯光色温
- 观察办公室的灯光色温
- 感受不同色温的氛围

**学习资源**：
- B站搜索"照度 色温 显色指数"
- 云知光照明学院（免费课程）

**检验**：
- [ ] 能用自己的话解释光通量、照度、亮度
- [ ] 能说出 3 种常见色温的特点
- [ ] 能说出显色指数的作用

### Day 2: 理论学习（周二）

#### 🌙 晚上（20:00-21:00）- 灯光设计

**任务 2.1: 学习办公空间照明标准**

**学习内容**：

**国家标准（GB 50034）**：
| 区域 | 照度要求 | 色温 | 显色指数 |
|------|---------|------|---------|
| 普通办公室 | 300 lx | 4000K | ≥80 |
| 设计室 | 500 lx | 4000K | ≥80 |
| 会议室 | 300 lx | 4000K | ≥80 |
| 接待区 | 200 lx | 3000K | ≥80 |
| 走廊 | 100 lx | 4000K | ≥60 |

**记忆方法**：
- 300 lx = 办公室标准
- 500 lx = 精细工作（设计）
- 4000K = 办公首选（中性）
- 80 = 最低显色指数

**任务 2.2: 学习照度计算方法**

**方法 1：利用系数法**
```
E = (N × Φ × U × K) / A
```
- E：平均照度 (lx)
- N：灯具数量
- Φ：单个灯具光通量 (lm)
- U：利用系数（一般 0.5-0.7）
- K：维护系数（一般 0.7-0.8）
- A：房间面积 (m²)

**简化版**：
```
N = (E × A) / (Φ × U × K)
```

**实际计算**：
- 房间：20 ㎡（4m × 5m）
- 要求照度：300 lx
- 灯具：筒灯 10W，900 lm
- 利用系数：0.6
- 维护系数：0.8

计算：
```
N = (300 × 20) / (900 × 0.6 × 0.8)
N = 6000 / 432
N = 13.9 ≈ 14 盏
```

**练习**：
- 练习 1: 30 ㎡办公室，300 lx，筒灯 10W（900 lm）
- 练习 2: 50 ㎡会议室，300 lx，筒灯 10W（900 lm）

**学习资源**：
- 《照明设计手册》
- 云知光照明学院

**检验**：
- [ ] 能记住办公空间照度标准
- [ ] 能用利用系数法计算灯具数量
- [ ] 完成 2 个计算练习

- 面积：20 ㎡（4m × 5m）

- 层高：3.6m
- 照度要求：300 lx
- 色温要求：4000K
- 显色指数：≥80

**任务**：
1. 计算需要的灯具数量
2. 选择合适的灯具
3. 布置灯具位置

**步骤**：
```
步骤 1: 计算灯具数量
N = (E × A) / (Φ × U × K)
N = (300 × 20) / (900 × 0.6 × 0.8)
N = 14 盏

步骤 2: 选择灯具
- 型号：筒灯 10W
- 光通量：900 lm
- 色温：4000K
- 显色指数：≥80
- 光束角：60°

步骤 3: 布置灯具
- 间距：2m × 2.5m
- 排列：3 行 × 5 列 = 15 盏
- 略一点余量，照度会稍高
```

**输出**：
