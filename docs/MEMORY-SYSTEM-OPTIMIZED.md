## 🎯 四种封闭类型应用

### user（用户画像）✅
- **目的**: 建立用户画像，个性化协作
- **存储内容**: 角色、技能、偏好
- **关键**: 避免负面判断

### feedback（正负反馈）✅
- **目的**: 记录用户对 AI 行为的指导
- **关键**: **正负反馈都要记录**（避免过度保守）
- **格式**: 规则 + Why + How to apply

### project（项目上下文）✅
- **目的**: 记录不可从代码推导的项目信息
- **关键**: **相对日期转绝对日期**
- **格式**: 目标 + Why + How to apply

### reference（外部指针）✅
- **目的**: 存储外部资源的指针
- **关键**: 存位置不存内容
- **格式**: 指针 + 描述 + 访问方式
