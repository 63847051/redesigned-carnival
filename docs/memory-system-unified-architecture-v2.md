## 🎯 **核心问题**

### ❌ **潜在的混乱**
- 8 个概念互相独立
- 没有统一的入口
- 数据可能重复
- 检索可能冲突

### ✅ **解决方案**
- **统一入口** - 所有操作通过统一接口
- **分层架构** - 清晰的职责划分
- **数据流向** - 明确的数据流
- **协调机制** - 自动协调避免冲突

## 🎯 **统一接口设计**

### ✅ **MemoryManager - 统一入口**

```python
class MemoryManager:
    """记忆系统统一管理器"""
    
    def __init__(self):
        # 初始化所有子系统
        self.filesystem = FileSystemContext()
        self.layered = LayeredContextLoader()
        self.retriever = RetrievalVisualizer()
        self.persistence = MemoryPersistenceHooks()
        self.coordinator = MemorySyncCoordinator()
    
    def save_memory(self, key: str, content: str, layer: str = "l1"):
        """
        保存记忆（统一入口）
        
        流程:
        1. 保存到文件系统范式
        2. 根据分层规则存储
        3. 记录检索轨迹
        """
        # 1. 保存到文件系统
        self.filesystem.add_memory(key, content)
        
        # 2. 根据分层规则存储
        self.layered.add_to_layer(layer, key, content)
        
        # 3. 记录检索轨迹
        self.retriever.record_retrieval(key, [{
            "source": "memory",
            "path": f"{layer}/{key}",
            "score": 1.0,
            "reason": "直接保存"
        }])
        
        return f"✅ 记忆已保存: {key}"
    
    def search_memory(self, query: str, method: str = "hybrid") -> List[Dict]:
        """
        检索记忆（统一入口）
        
        流程:
        1. 向量检索（QMD + semantic）
        2. 记录检索轨迹
        3. 返回结果
        """
        # 1. 向量检索
        if method == "hybrid":
            # QMD 全文搜索
            results = qmd_search(query)
            
            # 语义搜索
            semantic_results = semantic_search(query)
            
            # 合并结果
            results = merge_results(results, semantic_results)
        
        # 2. 记录检索轨迹
        self.retriever.record_retrieval(query, results, method)
        
        # 3. 返回结果
        return results
    
    def load_context(self, token_budget: int = 4000) -> str:
        """
        加载上下文（统一入口）
        
        流程:
        1. 分层加载（L0/L1/L2）
        2. Token 优化
        3. 返回上下文
        """
        return self.layered.load_context(token_budget)
    
    def sync_to_memory(self) -> str:
        """
        同步到长期存储（统一入口）
        
        流程:
        1. 检查一致性
        2. 同步 .context/ → memory/
        3. 返回报告
        """
        return self.coordinator.sync_to_memory()
```

## 🚨 **避免冲突的机制**

### ✅ **1. 单一入口原则**
- 所有操作通过 `MemoryManager`
- 不允许直接调用子系统
- 避免数据重复

### ✅ **2. 职责分离原则**
- 每个系统只负责一件事
- 不重复功能
- 清晰的边界

### ✅ **3. 数据流向原则**
- 写入：用户 → MemoryManager → 子系统
- 读取：用户 → MemoryManager → 子系统
- 同步：MemoryManager → 协调器

### ✅ **4. 自动协调原则**
- 每天自动同步
- 自动检查一致性
- 自动解决冲突
