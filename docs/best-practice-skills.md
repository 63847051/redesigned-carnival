## 🎯 **Skills 最佳实践**

### 📋 **什么是 Skill？**

Skill 是**注入到现有上下文的可配置知识**：
- ✅ 领域专业知识
- ✅ 工作流程指导
- ✅ 最佳实践
- ✅ 可配置和预加载

### 🎯 **何时使用 Skill？**

**适用场景**：
- ✅ **专业知识** - 需要领域专长
- ✅ **工作流程** - 标准化流程
- ✅ **最佳实践** - 经验总结
- ✅ **频繁使用** - 常用知识

**不适用场景**：
- ❌ 一次性任务
- ❌ 简单查询
❌ 动态内容

## 🚀 **创建 Skill**

### 方法 1: 通过文件创建

**位置**: `.claude/skills/<name>/SKILL.md`

**模板**：
```markdown
# <Skill Name>

## 概述
<技能的简短描述>

## 适用场景
- 场景 1
- �景 2
- 场景 3

3. 原则 3

## 工作流程
1. 步骤 1
2. 步骤 2
3. 步骤 3

## 最佳实践
- 实践 1
- 实践 2
- 实践 3

## 常见问题
- Q1: ...
- A1: ...

## 参考资料
- 资源 1
- 资源 2
```

### 方法 2: 通过对话创建

**在 Claude Code 中**：
```
> 创建一个技能，用于 <task>
```

**Claude 会自动**：
1. 创建技能目录
2. 编写 SKILL.md
3. 配置加载

## 🎯 **使用 Skill**

### 方法 1: 自动发现
Claude Code 会自动发现技能并加载

### 方法 2: 手动引用
在对话中引用：
```
> 使用 python-developer 技能优化这段代码
```

### 方法 3: 配置预加载
在 `.claude/settings.json` 中配置：
```json
{
  "skills": {
    "preload": ["python-developer", "frontend-developer"]
  }
}
```
