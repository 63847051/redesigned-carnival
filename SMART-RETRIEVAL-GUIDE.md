# 🚀 智能检索使用说明

## 🎯 两种检索方式

### **方式 1: 快速检索（ms.sh）**
```bash
cd /root/.openclaw/workspace
./ms.sh "查询" 3
```

**特点**：
- ✅ 超快（< 100ms）
- ✅ 免费（只用 QMD）
- ⚠️ 准确度一般

### **方式 2: 智能检索（ms2.sh）** ⭐ 推荐
```bash
cd /root/.openclaw/workspace
./ms2.sh "查询" 3
```

**特点**：
- ✅ 更准确（LLM 理解语义）
- ✅ 更智能（AI 排序）
- ⚠️ 稍慢（需要 2-3 秒）

---

## 📊 对比

| 特性 | ms.sh | ms2.sh |
|------|-------|--------|
| **速度** | ⚡⚡⚡ | ⚡⚡ |
| **准确度** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **成本** | 免费 | 低（LLM API） |
| **推荐场景** | 日常快速查找 | 重要准确查询 |

---

## 💡 使用建议

### **日常使用：ms.sh**
```bash
# 快速查找
./ms.sh "部署"
./ms.sh "防护"
```

### **重要查询：ms2.sh**
```bash
# 需要准确结果时
./ms2.sh "三重防护机制详解"
./ms2.sh "OpenCode CLI 正确用法"
```
